Quiz — Définir l'art

Nietzsche, Kant, Arendt, Duchamp…

Question 1

Pour Nietzsche, le génie est :

  • [ ] Un don divin
  • [✓] Le résultat d'un travail acharné
  • [ ] Une inspiration
  • [ ] Une nature

Explication. Critique du culte du génie : « aucune activité n'est un miracle ».

Question 2

Pour Arendt, les œuvres d'art sont :

  • [ ] Faites pour les hommes
  • [✓] Faites pour le monde, destinées à survivre
  • [ ] Des objets d'usage
  • [ ] Des biens de consommation

Explication. Gratuité de l'art : sans fonction dans le processus vital de la société.

Question 3

Selon Alain, l'artiste :

  • [ ] Suit un plan précis
  • [✓] Est aussi spectateur de son œuvre
  • [ ] Imite la nature
  • [ ] Reproduit des modèles

Explication. « L'idée lui vient à mesure qu'il fait. » L'artisan, lui, suit un plan.

Question 4

Pour Kant, le beau est :

  • [ ] Objectif
  • [✓] Subjectif mais universel
  • [ ] Relatif
  • [ ] Personnel

Explication. « Le beau est ce qui plaît universellement et sans concept. »

Question 5

L'agréable, selon Kant :

  • [ ] Plaît à tous
  • [✓] Procure un plaisir sensoriel particulier
  • [ ] Est désintéressé
  • [ ] Est universel

Explication. L'agréable dépend des goûts de chacun ; le beau prétend à l'universalité.

Question 6

La Fontaine (1917) est de :

  • [ ] Picasso
  • [✓] Duchamp
  • [ ] Manzoni
  • [ ] Warhol

Explication. Marcel Duchamp : un urinoir signé, exposé comme œuvre d'art (ready-made).

Question 7

La caractéristique universelle de l'art est, dans la conclusion de la séquence :

  • [ ] La beauté
  • [ ] Le génie
  • [✓] La défamiliarisation
  • [ ] La technique

Explication. L'art rend les choses étranges pour court-circuiter le regard habituel.

Question 8

Pour Aristote, l'art :

  • [ ] Crée la beauté
  • [✓] Permet de contempler ce qui dans la réalité serait pénible
  • [ ] Imite les dieux
  • [ ] Doit être utile

Explication. Plaisir esthétique antique : contemplation de l'ordre, même dans le répugnant.
