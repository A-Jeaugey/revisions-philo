# Quiz — Devoir et bonheur

> Calliclès, Épicure, stoïcisme…

### Question 1

Quelle est la différence entre obligation et contrainte ?

- [ ] Aucune
- **[✓]** Obligation = libre, contrainte = extérieure
- [ ] Obligation = forcée
- [ ] Contrainte = morale

*Explication.* L'obligation s'impose librement (morale) ; la contrainte est extérieure et forcée.

### Question 2

Pour Calliclès, le bonheur consiste à :

- [ ] Maîtriser ses désirs
- **[✓]** Satisfaire compulsivement tous ses désirs
- [ ] Méditer
- [ ] Aider les autres

*Explication.* Hédonisme radical : « vivre selon la nature ».

### Question 3

Le sort des Danaïdes illustre :

- [ ] Le devoir moral
- **[✓]** La frustration de la vie immodérée
- [ ] L'amour divin
- [ ] Le travail

*Explication.* Remplir des tonneaux sans fond = la vie des plaisirs sans cesse renouvelés.

### Question 4

Pour Épicure, le bonheur est :

- [ ] Le plaisir intense
- **[✓]** L'ataraxie
- [ ] La gloire
- [ ] La richesse

*Explication.* Ataraxie = absence de souffrance physique et psychologique.

### Question 5

L'épicurisme est :

- [ ] Un hédonisme radical
- **[✓]** Un hédonisme modéré par la raison
- [ ] Un ascétisme
- [ ] Un stoïcisme

*Explication.* Satisfaire seulement les désirs naturels et nécessaires.

### Question 6

Le stoïcisme propose :

- [ ] D'agir sur le monde
- **[✓]** D'agir sur soi-même
- [ ] D'éviter le monde
- [ ] De rechercher le plaisir

*Explication.* Bonheur absolu, indépendant des circonstances extérieures.

### Question 7

L'étymologie de bonheur est :

- **[✓]** Bon + heur (chance)
- [ ] Bien + cœur
- [ ] Bon + jour
- [ ] Bien + heure

*Explication.* « heur » = chance — d'où le problème : peut-il y avoir un devoir d'être heureux ?
